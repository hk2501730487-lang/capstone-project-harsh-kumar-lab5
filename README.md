## Short description
A responsive static landing page demonstrating semantic HTML5, responsive CSS3 using Flexbox & Grid, accessible forms, course cards, testimonials, and contact form.

## Sections implemented
- Header / Navigation (mobile toggle)
- Hero/banner with CTA
- Featured Courses (CSS Grid)
- Filters (UI only)
- Testimonials
- Contact / Sign-up form
- Footer

## Breakpoints & layout techniques
- Breakpoints: 479px (small phone), 768px (tablet), 1024px (desktop)
- Layout: Flexbox for header/footer alignment, CSS Grid for course cards & hero layout
- Typography: Google Fonts (Inter), relative units, consistent weights
- Visuals: gradient buttons, shadows, hover transitions
